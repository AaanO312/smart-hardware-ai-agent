扫地机器人故障排查知识库

一、无法回充

可能原因：
1. 充电座未通电。
2. 充电座位置不合适。
3. 充电触点脏污。
4. 机器人地图定位异常。
5. 回充路径被障碍物阻挡。
6. 电池电量过低或电池异常。

排查步骤：
1. 检查充电座电源指示灯是否正常。
2. 保持充电座左右各 0.5 米、前方 1.5 米范围内无遮挡。
3. 擦拭机器人和充电座的金属触点。
4. 将机器人放回充电座重新出发。
5. 如果近期移动过充电座，建议重新建图。
6. 如果多次无法回充，应联系售后检测电池或传感器。

二、地图丢失或地图错乱

可能原因：
1. 清扫过程中人为搬动机器人。
2. 移动了充电座位置。
3. 室内布局发生大幅变化。
4. 清扫中途断电。
5. 传感器被灰尘遮挡。
6. 软件或固件异常。

处理建议：
1. 尽量不要在清扫过程中搬动机器人。
2. 充电座应固定摆放，不要频繁移动。
3. 擦拭激光雷达、摄像头和沿墙传感器。
4. 在 App 中尝试恢复历史地图。
5. 如果地图严重错乱，建议删除旧地图后重新建图。
6. 定期更新固件，提升建图稳定性。

三、清扫效果变差

可能原因：
1. 尘盒已满。
2. 滤网堵塞。
3. 主刷缠绕毛发。
4. 边刷变形或磨损。
5. 吸力模式过低。
6. 地面存在大颗粒垃圾或湿垃圾。
7. 风道堵塞。

排查步骤：
1. 清空尘盒。
2. 清理或更换滤网。
3. 拆下主刷，清除缠绕毛发。
4. 检查边刷是否变形，必要时更换。
5. 对灰尘较多或宠物毛发较多区域使用强力模式。
6. 检查吸尘口和风道是否堵塞。
7. 避免让机器人清理大量积水、黏性污渍或大块垃圾。

四、拖地效果差

可能原因：
1. 水箱缺水。
2. 拖布过脏。
3. 出水量设置过低。
4. 拖布安装不到位。
5. 地面污渍过重。
6. 水箱出水孔堵塞。

处理建议：
1. 检查水箱水量。
2. 清洗拖布并重新安装。
3. 根据地面材质调整出水量。
4. 木地板建议使用低水量或标准水量。
5. 厨房油污、干结污渍需要人工预处理。
6. 定期清理水箱出水孔。

五、机器人频繁卡住

可能原因：
1. 地面电线、袜子、玩具较多。
2. 地毯边缘过高。
3. 家具底部高度接近机身高度。
4. 门槛过高。
5. 桌椅腿结构复杂。

处理建议：
1. 清扫前整理地面杂物。
2. 对复杂区域设置禁区或虚拟墙。
3. 对容易卡住的地毯边缘进行固定。
4. 门槛较高时不建议强行跨越。
5. 如果同一区域反复卡住，可在 App 中设置避让区域。

六、无法连接 WiFi

可能原因：
1. WiFi 密码错误。
2. 路由器使用 5GHz 网络，设备只支持 2.4GHz。
3. 网络信号弱。
4. App 权限不足。
5. 设备处于非配网模式。
6. 路由器开启了特殊限制。

处理建议：
1. 确认使用 2.4GHz WiFi。
2. 检查 WiFi 密码。
3. 将机器人靠近路由器重新配网。
4. 重置机器人网络配置。
5. 检查 App 是否开启蓝牙、定位和本地网络权限。
6. 如仍无法连接，可尝试重启路由器和机器人。

七、噪音变大

可能原因：
1. 主刷缠绕异物。
2. 边刷刮擦地面或变形。
3. 尘盒或滤网安装不到位。
4. 风道进入硬物。
5. 轮组卡入异物。
6. 强力模式本身噪音较高。

处理建议：
1. 检查主刷、边刷和轮组。
2. 清理风道和吸尘口。
3. 重新安装尘盒和滤网。
4. 夜间清扫建议使用静音模式。
5. 如果出现异常尖锐噪音，应停止使用并联系售后。

八、安全风险

以下情况应停止使用并联系售后：
1. 机器人进水后无法正常工作。
2. 电池明显鼓包。
3. 充电时出现异味、过热或冒烟。
4. 机身有明显破损并影响运行。
5. 机器人反复报错且无法通过重启恢复。
